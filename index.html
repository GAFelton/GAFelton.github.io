Weather Dashboard

Created a webpage to display current weather conditions and 5 day forecast based on user input. Used the 3 aspects of the OpenWeatherMap API: Current Conditions, UV Index, and the 5 Day Forecast. Stored previous searches in local storage, and dynamically populated a list of previous searches.

    Achievements:
  • All elements except for the search bar are dynamically generated using jQuery.
  • Page loads with most recent previous search, and the search list will collapse under a button.
  • Accounted for user search edge cases, such as 404 errors or searching for the same city twice.
Tools Used:
  • HTML
  • CSS / Bootstrap
  • JavaScript / jQuery
APIs Used:
  • OpenWeatherMap API:
    • Current Conditions
    • UV Index
    • Five Day Forecast
